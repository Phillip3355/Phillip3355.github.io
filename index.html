이미지 삽입 예제

다음 예제를 `stime.html`에 그대로 넣거나, 각 이미지를 실제 파일/경로로 교체하세요.

1) 로컬 이미지 (같은 폴더)

`image.jpg`를 `stime.html`과 같은 폴더에 두세요.

2) 서브폴더

`images/photo.jpg`와 같이 서브폴더를 사용합니다.

3) 외부 URL (예제 이미지)

웹에 있는 이미지를 바로 불러옵니다.

4) 데이터 URI (작은 아이콘)

작은 SVG 아이콘을 인라인으로 넣는 예시입니다.

접근성 & 성능 팁

  • 의미 있는 alt 텍스트를 제공하세요. 장식용이면 alt="" 사용.
  • 레이아웃 안정화를 위해 width와 height 속성을 지정하세요.
  • 지연 로딩은 loading="lazy"로 성능 개선.
  • 반응형 이미지는 CSS에서 max-width:100%; height:auto; 적용.

원하시면 다른 이미지로 바꿔드리거나, 파일 업로드 방법을 도와드릴게요.
